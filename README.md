# Iris Flower Classification using Logistic Regression

## 📌 Project Overview

This project demonstrates the implementation of Logistic Regression using the famous Iris Flower Dataset.  
The model is trained to classify iris flowers into different species based on flower measurements.

---

## 🚀 Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## 📂 Dataset Information

The project uses the Iris Flower Dataset, a well-known dataset in Machine Learning.

### Features

| Feature Name | Description |
|--------------|-------------|
| Sepal Length | Length of sepal in cm |
| Sepal Width  | Width of sepal in cm |
| Petal Length | Length of petal in cm |
| Petal Width  | Width of petal in cm |

### Target Classes

- Setosa
- Versicolor
- Virginica

### Dataset Details

- Total Samples: 150
- Number of Features: 4
- Number of Classes: 3

---

## 🤖 Machine Learning Algorithm

- Logistic Regression:
Logistic Regression is a supervised Machine Learning algorithm used for classification problems.
It predicts the probability of a data point belonging to a particular category or class.
